## Number Guessing Game
 Project Description

This is a simple interactive Number Guessing Game built using Python and Streamlit.

The app generates a random number between 1 and 100, and the user tries to guess it.
The app provides feedback if the guess is too high or too low and shows the number of attempts once the correct number is guessed.
